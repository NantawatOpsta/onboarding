# Django Workshop

ในเวิร์กช็อปนี้ เราจะเรียนรู้การพัฒนาเว็บแอปพลิเคชันด้วย Django โดยใช้แนวคิด Clean Architecture เพื่อให้โค้ดของเรามีความยืดหยุ่นและง่ายต่อการบำรุงรักษา

![An example image](clean-architect.png)

## เนื้อหาที่จะได้เรียนรู้

- การติดตั้งและตั้งค่า Django
- การสร้างโปรเจกต์และแอปพลิเคชัน Django
- การออกแบบฐานข้อมูลด้วย Django ORM
- สร้าง use cases เพื่อแยก business logic ออกจาก views
- สร้าง unit tests สำหรับ use cases

## จุดประสงค์ของโปรแกรม

จุดประสงค์ของโปรแกรมนี้คือการเรียนรู้การพัฒนาเว็บแอปพลิเคชันด้วย Django โดยใช้แนวคิด Clean Architecture เพื่อให้โค้ดของเรามีความยืดหยุ่นและง่ายต่อการบำรุงรักษา

- ไม่จำเป็นต้องสร้าง Views และ API
- สร้าง Model Product เพื่อจัดการสินค้า (Product Management System) ที่สามารถเพิ่ม แก้ไข ลบ และแสดงรายการสินค้าได้
- สร้าง Model Cart เพื่อจัดการตะกร้าสินค้า โดยสามารถเพิ่มสินค้าไปยังตะกร้า และแสดงรายการสินค้าที่อยู่ในตะกร้าได้
- สร้าง Model Promotion สำหรับจัดการการลดราคาสินค้าภายในระบบ

## Use Cases

การลดราคาของระบบ ใช้ model promotion ที่อยู่ในระบบโดยสามารถสร้าง condition ให้ได้ดังนี้

- เมื่อสินค้าอยู่ในตระกร้าสินค้าครบ 10 ชิ้น จะลดราคา 10%
- เมื่อสินค้าอยู่ในตระกร้าสินค้าครบ 15 ชิ้น จะลดราคา 15%
- เมื่อสินค้าอยู่ในตระกร้าสินค้าครบ 1000 บาท จะลดราคา 10%
- เมื่อสินค้าอยู่ในตระกร้าสินค้าครบ 2000 บาท จะลดราคา 20%

condition นี้จะต้องสามารถกำหนดได้จาก django ui
